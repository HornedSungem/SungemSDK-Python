## 安装说明

支持的平台:

- Linux
- macOS

支持的架构:

- i386 / i586 / i686 / x86_64
- armv6l / armv7l / aarch64

在终端运行如下命令来配置 `SungemSDK` 的运行环境：

- Linux:
    
    ```
    cd installer/Linux && ./install.sh
    ```

- macOS:

    ```
    cd installer/macOS && ./install.sh
    ```
